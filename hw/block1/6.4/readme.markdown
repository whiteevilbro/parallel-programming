# Task 6.4 (Working with JPF)

## Description

Solve [Task 1.3](https://github.com/Svazars/parallel-programming/tree/main/hw/block1/1.3) using tireless automaton, saving your own brain power.

## Requirements

Provide source code for your solution and corresponding JPF logs that clearly enumerate all possible results of concurrent program.
Use `target.args` to run your program under JPF in different modes:
- First mode: enlist all possible outcomes (no interleaving logs, faster exploration)
- Second mode: print interleaving logs for particular outcomes (those logs will be quite short and contain only few duplicates)
